What is this site?

Hi, and thank you for showing interest in this microsite!

"Asteroid of the Day" was built as a school project in the Front-End-Development program at Noroff. Inspired by the APoD (Astronomy Picture of the Day), I wanted to use NASA's NeoWs (Near Earth Object Web Service) to create this microsite to display some cool facts about asteroids. I'm a science enthusiast myself and I hope you enjoy the content of this site as much as I do.

All Near Earth Object data is retrieved from NASA's API's. For any questions or comments/feedback about the site, please contact me.
