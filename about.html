What is this site?

Hi, and thank you for showing interest in my little fun-site!

"Asteroid of the Day" was built as a school project in the Front-End-Development program at Noroff where the task was to create a microsite for NASA. Inspired by the APoD (Astronomy Picture of the Day), I wanted to use NASA's NeoWs (Near Earth Object Web Service) to display some cool facts about asteroids. As a space enthusiast I had a lot of fun creating this site. I'm also looking forward to maintain it, so if you have any questions or comments/feedback about the site, please don't hesitate to contact me.

All Near Earth Object data is retrieved from NASA's API's.

I hope you enjoy the content of the site as much as I do!

Jenny
